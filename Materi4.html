Menghitung Jumlah Besar Sudut Suatu Bangun

PETA KONSEP

1. Jumlah Besar Sudut pada Bangun Segi Tiga

Jumlah besar ketiga sudut pada bangun segitiga adalah 180°.

∠A + ∠B + ∠C = 180°

Contoh :

Tentukan besar ∠F pada bangun segitiga berikut!

Penyelesaian :

∠F + ∠G + ∠H = 180°

∠F + 45° + 90° = 180°

∠F = 180° - 90° - 45° = 45°

Jadi besar ∠F = 45°

2. Jumlah Besar Sudut pada Bangun Segi Empat

Jumlah besar keempat sudut pada bangun segiempat adalah 360°.

∠O + ∠P + ∠Q + ∠R = 360°

Contoh :

Tentukan besar ∠F pada bangun segi empat berikut!

Penyelesaian :

∠E + ∠F + ∠G + ∠H = 360°

135° + ∠F + 45° + 135° = 360°

∠F = 360° - 135° - 45° - 135° = 45°

Jadi besar ∠F = 45°

3. Jumlah Besar Sudut Pada Bangun Segi Banyak

Menentukan jumlah sudut pada bangun segi banyak dapat ditentukan dengan cara menggunakan busur derajat. Selain itu, ada juga rumus yang dapat digunakan untuk mencari jumlah sudut pada segi banyak. Adapun rumusnya yaitu :

Jumlah besar sudut segi-n = (n - 2) x 180

Dengan n adalah banyak sisi pada bangun segi banyak.

Contoh :

• Jumlah besar sudut segitiga = (3 - 2) x 180° = 1 x 180° = 180°

• Jumlah besar sudut segi empat = (4 - 2) x 180° = 2 x 180° = 360°

• Jumlah besar sudut segi lima = (5 - 2) x 180° = 3 x 180° = 540°

Penyelesaian :

∠R + ∠S + ∠T + ∠U + ∠V = 540°

Ringkasan

• Jumlah besar ketiga sudut pada bangun segitiga adalah 180°

• Jumlah besar keempat sudut pada bangun segiempat adalah 360°

• Rumus menentukan Jumlah besar sudut pada bangun segi n :

Jumlah besar sudut segi-n = (n - 2) x 180°

2. Berapakah jumlah besar sudut pada bangun segi lima?

3. Bagaimana cara menghitung jumlah besar sudut jika tidak dapat menggunakan rumus?

Kunci Jawaban Latihan 4

1. besar ∠Y pada bangun segi tiga tersebut adalah

∠X + ∠Y + ∠Z = 180°

∠30° + ∠Y + 110° = 180°

∠Y = 180° - 30° - 110° = 40°

Jadi besar ∠Y = 40°

2. Besar ∠P pada bangun segi empat tersebut adalah

∠O + ∠P + ∠Q + ∠R = 360°

45° + ∠P + 65° + 110° = 360°

∠P = 360° - 45° - 65° - 110° = 140°

Jadi besar ∠P = 140°

3. Jumlah besar sudut pada bangun segi enam adalah = (6 - 2) x 180° = 4 x 180° = 720°

4. Jumlah besar sudut pada bangun segi tujuh adalah = (7- 2) x 180° = 5 x 180° = 900°

5. Jumlah besar sudut pada bangun segi delapan adalah = (8 - 2) x 180° = 6 x 180° = 1080°
